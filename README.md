
### 使用方式

* 启动镜像


```shell
$ ./start-docker.sh
```

* 调试镜像

```shell
$ ./debug-docker.sh
```

* 停止镜像

```shell
$ ./stop-docker.sh
```

* 清理镜像

```shell
$ ./clean-docker.sh
```

* 测试镜像

```shell
$ ./test-docker.sh
```
